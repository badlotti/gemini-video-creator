# 名探偵シャノワールと消えた黄金の魚

## あらすじ
夜の街を舞台に、クールな猫の探偵「シャノワール」が、お嬢様猫から盗まれた「黄金の魚」の捜索依頼を受ける。わずかな手がかりを頼りに犯人を追い詰めるが、その先には意外な真実が待っていた。ミステリーと可愛らしさが融合した、ハートフルなショートアニメーション。
